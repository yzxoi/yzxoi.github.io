10086. 「一本通 3.3 练习 3」Easy SSSP

yzxoi
2019-02-15 (Updated: 2022-06-25)
oi
LOJ, spfa

题意

给你一个图，问从源点到每个节点的最短路径分别是多少。 如果存在负权回路，只输出一行 -1；如果不存在负权回路，再求出一个点 S 到每个点的最短路的长度。如果 S 与这个点不连通，则输出 NoPath。

思路

当然食用spfa啦。 先跑一下非源点的。万一数据卡你其他有环呢？ 然后再跑一次源点。得出Ans
