位运算, 数位dp

题目链接:bzoj Luogu

Description

求$1$到$n$每个数的$1$的个数之积。

对于$ 100\% $的数据，$n\leq 10^15$

Solution

数位$DP$。

可以先把$n$拆成二进制，然后$DFS$。
