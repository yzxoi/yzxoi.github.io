线段树

Preface

题目链接

Description

给定一个序列A[i]，每次询问l,r，求[l,r]内最长子串，使得该子串为不上升子串或不下降子串

N,Q<=50000

Solution

设 $L[i]$ 表示以 $i$ 为起点，往左最多能成为子串的长度，$R[i]$ 表示以 $i$ 为起点，往右最多能成为子串的长度。

那么很显然区间 $[l,r]$ 答案就是 $\max\{R[l],R[l+1],\dots,R[r-L[r]],L[r]\}$（处理边界）

那么再套一个线段树就好了
